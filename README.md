# to_voice
将网页或微信文章装换

使用讯飞语音的 [Web API 接口][1]，实现将文字转化为语音文件。

# 类似产品

[百度广播开放平台][2]

[在线免费文字转语音系统][3]


大部分都是在线输入文字或者 URL，然后生成音频文件。

## 功能

声音可以选择：

- 情感女声
- 情感男声
- 非情感女声
- 非情感男声

语速有三挡可选：

- 慢速
- 正常
- 快速
